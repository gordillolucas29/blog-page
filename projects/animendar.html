Animendar

Origen.
"Animendar" fué llevado a cabo para poder tener una forma más rápida y versátil de ver que serie salía x día.

Herramientas.
Este proyecto se desarrolló en Visual Studio Code y fué realizado mediante la implementación de conceptos que llevaba arraigando de HTML y CSS.

En el transcurso de la realización de este mismo, se fueron implementando mejoras para el confort del usuario y la calidad de la página.
Esto sumó la implementación del uso de un dominio gratuito gracias a Freenom
El uso de GitHub para almacenar el proyecto y subirlo a "github-pages" (Repo en GitHub).

El sitio web puede ser visitado en el siguiente enlace: animendar.ml

Algunas de estas mejores fueron principalmente:

Una nueva paleta de colores.
En un principio el sitio web contaba con una gamma de colores beige. Sin embargo estos no terminaban de complementarse adecuadamente, de éste modo no dando el mejor resultado en lo que apariencia respecta. Tomando esto en cuenta fué como fué implementada una nueva paleta de colores, así de este modo, cada color correspondería a otro dando una apariencia muchísimo mejor al sitio.

Paleta implementa en el sitio web.



Sitio web responsive.
Para dar un salto de confort y calidad para el usuario, fué implementado el responsive web en este sitio, así de éste modo poder visualizar la página web desde cualquier dispositivo sería cómodo, fácil y rápido.

Esta implementación se dió utilizando media querys en CSS.
